1. CALCULATOR
This project is made using three concepts - Ternaries, 
Switch-Cases and If-Else statements.
The calculator solves te following, mentioned along with
the calling characters:
Sum - '+'
Product - '*'
Difference - '-'
Dividend - '/'
Compund Interest - 'c'
Simple Interest - 's'
Multiplication of Matrices - 'X'
Profit/Loss - '!'
